# Users and Groups

## Objective

Document basic Active Directory user and group management concepts for Helpdesk and IT Support practice.

## Example Users

| Username | Department |
|---|---|
| john.smith | HR |
| sarah.johnson | Sales |
| michael.brown | IT |
| helpdesk.user | IT Support |

## Example Groups

| Group Name | Purpose |
|---|---|
| HR_Users | HR department users |
| Sales_Users | Sales department users |
| IT_Users | IT department users |
| Helpdesk_Users | Helpdesk support users |

## Basic Workflow

1. Open Active Directory Users and Computers.
2. Select the correct Organizational Unit.
3. Create a new user account.
4. Assign username and temporary password.
5. Enable "User must change password at next logon".
6. Add the user to the correct security group.
7. Document the action in the ticket.

## Helpdesk Relevance

User creation and group assignment are common IT support tasks. Group membership helps control access to shared folders, applications and company resources.
