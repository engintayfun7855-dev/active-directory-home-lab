# Active Directory Home Lab

This repository documents my Active Directory learning lab for entry-level IT Support, Helpdesk and Desktop Support roles.

The purpose of this project is to practice and document basic Windows Server and Active Directory tasks such as user management, group management, password reset, account unlock and basic Group Policy concepts.

## Lab Status

In progress.

## Topics Covered

- Windows Server basics
- Active Directory Domain Services
- Domain Controller concept
- User account management
- Security groups
- Organizational Units
- Password reset workflow
- Account unlock workflow
- Basic Group Policy concepts
- Helpdesk troubleshooting notes

## Planned Lab Environment

| Component | Description |
|---|---|
| Server OS | Windows Server |
| Client OS | Windows 10 / Windows 11 |
| Domain Example | lab.local |
| Server Name | DC01 |
| Tool | Active Directory Users and Computers |

## Why This Project Exists

This project was created to demonstrate practical IT support knowledge for entry-level Helpdesk, Desktop Support and Junior IT Support positions.

The focus is not only on technical configuration, but also on documenting support workflows clearly.
